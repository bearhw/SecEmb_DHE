
== TRAIN directory 

DATASETS directory has instruction for training dataset preparation.

TRAIN directory has scripts for training table, DHE uniform and DHE varied.
(Rename desired DHE to proper dhebag.py)
Also has script for evaluating trained model accuracy.

CONVERT_DHE2LS directory has scripts for caching DHE as a table representation.
We provide scripts for uniform only (hybrid can be done similarly).

== PROFILER_INFERENCE directory

Has scripts for profiling, threshold calculation and end-to-end inference.
